# **App Name**: CareerCompass AI

## Core Features:

- ATS Resume Analyzer: Analyzes the resume against a given job description, providing feedback on keyword matching and formatting optimized for ATS systems. Uses generative AI tool for comparing the resume with Job description and suggest improvement and provide ATS score
- LaTeX Resume Builder: Generates a resume using LaTeX based on user input through a form. Users can download both the generated PDF and the LaTeX code.
- Career Roadmap Generator: Generates a personalized career roadmap based on user-selected career path, current skills, and desired timeline, providing stage-wise learning resources and practice project ideas. Provides links to external learning and recommends courses and material for free.Uses generative AI tool for creating steps in roadmap
- Social Account Analysis: Analyzes user profiles from platforms like LeetCode, GitHub, and Stack Overflow to assess skills and experience.
- Consolidated Scoreboard: Presents a unified score based on the resume analysis, career roadmap alignment, and social account analysis to give a comprehensive evaluation of the user's profile.
- Job Recommendation Engine: Recommends relevant job openings based on user's skills, experience, and the consolidated scoreboard results, sourced from platforms like LinkedIn, Glassdoor, and Foundit.

## Style Guidelines:

- Primary color: Strong, professional blue (#2962FF) to reflect trustworthiness and competence. Based on the look and feel of MyJob, this color echoes the company's branding. Also avoid to literally copy MyJob.
- Background color: Light, desaturated blue (#E6F0FF) to provide a clean and calming backdrop that reduces visual fatigue.
- Accent color: Warm orange (#FF9100) to highlight key actions and recommendations, drawing user attention to critical areas. The color orange is also analogous to blue, offering a more vibrant experience.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a clean and modern aesthetic, with 'Source Code Pro' used for code snippets.
- Use a set of professional and easily recognizable icons to represent different sections and functionalities, inspired by design principles used in job portal websites, for clarity and ease of use.
- Employ a clean, structured layout with clear visual hierarchy to facilitate easy navigation and readability.
- Subtle transitions and loading animations to improve user experience without overwhelming the user.
